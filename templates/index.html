What You Need to Know

MetaLab is a Data extraction tool that can be used to extract data quickly and effectively while providing accurate information about the data being extracted.

With this free online tool you can extract metadata from files of arbitrary type. Metadata might contain the name and login of the author, the creation date or other interesting details. At the moment we support the following file types:

Frequently Asked Questions

Do I need to pay to use Metlab?
Nope! Up to five usage of Metlab per day are completely free for each user. There is no need for a license and subscription.
What is metadata?
Metadata is basically a set of data that describes and gives information about other data which could be different file types such as audio, image, PDF, CSV etc.
Will you be adding new tools to Metlab in future?
Yes, we always look for ways to improve our product for a better user experience by carrying out various researches.
Why is metadata important?
Metadata is beneficial to almost every person or organisation that makes use of files as they would at some point want to find out basic information about their files for different purposes.
